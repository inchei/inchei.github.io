僅僅是因為[ S1 吹樓中某位病友一時興起的提議](https://bbs.saraba1st.com/2b/forum.php?mod=viewthread&tid=1336553&page=714#pid46626561)，我在 GitHub 上建了這樣一個項目：[《京吹學報》](https://github.com/hibikilogy/hibikilogy.github.io)，瀏覽地址是：https://hibikilogy.github.io。

這是一個搭載於 gh-page 的 Jekyll 項目，目的是收錄最近大熱的吹學著作——人一旦在封閉的環境待得久了，真是什麼都能創造出來。從《[為什麼歷史選擇了久石奏](https://tieba.baidu.com/p/6476296121)》到《[旗幟鮮明反對〈如何問責〉中的部分觀點--傘木希美之我見](https://bbs.nga.cn/read.php?tid=20651882)》，一個接一個令我目瞪口呆、由衷佩服。然而對於這些文章，由於刻意的政治風格，加之其中平台之一的 [NGA](http://bbs.nga.cn/thread.php?fid=-447601) 經常沖水，姑且先這樣把它們儲存起來。

雖然用的是早已折騰過的 Jekyll，但還是搞了半個多小時才做完。與此同時，還算是收穫頗豐——總算把之前從未正經看過的 git 文檔給掃了一遍，感歎自己之前對 git 的理解是多麼幼稚。

這個項目比起技術性較強的來説，更容易執行，所以這幾天打算儘量把幾篇文章給集齊了；有空想辦法借寫快速根據原帖生成 markdown 的工具來練練手。

<del>最後，希望吹學繼續繁榮昌盛！</del>
