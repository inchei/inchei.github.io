一方面出於測試一下 [Selenium](https://selenium.dev/) 以曲線救國式地完成 [NGA](https://bbs.nga.cn/) 爬蟲，另一方面也不能忍受 Edge Insider 在暫時借用的母親的小電腦上學着它肚子裏的 Chrome 瘋狂吃內存，正式從 chromium 換到了 Firefox。

本來想着只用來作爬蟲用的，但試用的時候認真思考了一下 Firefox 的優點。
- 比起 Edge Insider 不得不偽裝成 Chrome，Selenium 可以直接使用。
  - 最開始的原因。
- 在這麼一台機子上，Firefox **省內存**的功力的確體現了出來。
- **真的比 Chrome UI 好看**。設計癖又出來了。
- 自帶了視頻小窗口的功能——雖然這一點 Chrome 用插件也能做到，但方便。
- 附加組件商場的訪問對於仍然沒有穩定翻牆方式的人來説快多了（但説實話還是很慢），而且從外部加載插件再也不用忍受安全提示了。
  - 不過相對來説插件比 Chrome 少……
- 據説曾經説自己是最後一個獨立瀏覽器，有點興趣。
- 據説自由度比較高，還沒感受到。
- 貌似 Linux 上 Firefox 的佔有率比較高。説起來沒有空餘的機子給我玩 Linux 了，唉。

總而言之遷移到 Firefox 了，今後還得仔細研究一下這個傳奇瀏覽器。
