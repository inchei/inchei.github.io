中考結束了。考完化學的那一刻，我簡直想直接從椅子上蹦起來，然後放聲大笑。可是敢於這樣在公衆面前蛇精病的年齡已經過去了，所以我就使勁用手捏著臉，企圖控制住它上面要迸裂出來的笑容。同時，心裏也在突然間升起了懷疑：等到六月二十九日，我還能這麽笑著嗎？

想起了 S1 [吹吹病棟](https://bbs.saraba1st.com/2b/thread-1336553-1-1.html)裏面關於傘哥哥所謂「沒心沒肺的笑」的有趣<del>迫害</del>言論。又想笑自己，爬了好幾周吹吹樓，卻以中考為「藉口」連《利兹與青鳥》還沒有真正看過；但是我的心中確實是有了傘木希美的，那并不是武田綾乃或者山田尚子或者 S1 扭曲怪們的傘木，是我在心裏從他人口述裏捏造的一個形象——自然是扭曲的了。我最喜歡的就是「沒有才能」這一點，仿佛在撕裂自己的傷口似的喜歡。於是《利兹與青鳥》就變成了我的「金閣寺」。明天打算把這部從剛上映拖到現在的電影補掉，在那之後，我的傘木就再也沒有了。

與此同時要消失的，還有 JK 這個神奇的名詞。我永遠不曾想到過，自己會是一個自己曾經祇想遠觀而不願成爲的高中生，即便在幾個月前還在學習高中課程的時候也是如此；不久以前，我甚至還考慮著在十九歲時因爲什麽事故死去——這當然是無責任的，但長大似乎就不得不從「無責任」到「有責任」。我從未覺得高中生如何，可是當我自己成爲高中生，我便越發覺得自己的「衰老」。再也沒有少年天才，再也沒有恃才跋扈。學校沒有競賽的經驗，我們的班級是第一代；於是曾以爲必將走上的那些陽光大道，全都一個接一個黯淡下去，留下的祇有名爲「高考」的一條荊棘之路。

<aside>説來慚愧，中考時還以這個為藍本寫的作文，不過不同的是在作文裏我仍是一名運動員，還不是作爲我從小憧憬的二傳手，而是一名主攻手；畢竟還是扣球時的心理容易描寫。</aside>

<i>這是必然的</i>，我想。首先擺在我面前的是一條運動員的路，但是我卻因爲一次跌倒回頭而行；然後是和老校的招生時間擦肩而過；在外來變故的影響下，也沒能追上同年九月份前生的西交大少年們的脚步；最近一次，還是戰戰兢兢沒有放棄提前學習高中課程，去像由崎星空一樣用迷之自信挑戰學區外的名校好班。

我以爲我的青春這樣就祇能失去光芒了。巧合，偷窥了多年前賽老闆的博客，竟然正是从初三開始写的。

> 其实博客就像赛老板那样，只为自己而写、而看不就够了吗？每个人终究只是在写着自己，而一个过路人的偷窥，也是在他人中寻找自己。所以不必想着写什么，只要写，写就行了。这个想法，好像和前几天看完的《病隙碎笔》里的又相似了。
> 高中继续寄宿制，没法以赛老板一样的频率更新博客。不过这个博客衰微的年代里，还有着楼上那样无数的一篇博文式博客呢。只要自己能一直抱着这种心态写博客，我就满足了。

腦子一熱，在班固米留下了這樣的話。受其感觸，甚至想再建一個 hexo 主題：那種單頁的、簡單得要死的甚至時間也不要的博客；不過還是形式而已，先在 GitHub 上建了個[徒有表面的倉庫](https://github.com/inchei/hexo-theme-single)，慢慢來。很囧的一件事是，上回把 hexo 源文件傳到 GitHub 上時竟整個主題文件夾沒有傳上去，祇好重新再改一回主題。

不管是博客還是生活，這個「尋找自己」的説法，異乎尋常地符合我的狀態。回頭，又看了一遍[釘子中三時的博文](http://blog.dimpurr.com/fly-out/)——不經意發現他已經許久不上博客了。無論是傘木、賽老闆或者釘子，這些我腦中的形象正在我眼前照明我的路——沒有鮮花、沒有掌聲的，一條狹長的路。
